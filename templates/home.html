How to install and play:
  1. In powershell, cd to the game directory
  2. Run 'python app.py [name]' in the console, replacing [name] with your name
  3. Install any python modules that you may be missing using 'pip install [module], replacing [module] with whatever module the console says you're missing
  4. Repeat steps 2 and 3 until you're no longer missing any modules
  5. Use the number keys to place plants and survive for as long as you can!
Plant Appearance Description
Pea Shooter It shoots peas!
Snow Pea Shoots peas that freeze zombies.
Sunflower Periodically generates gold.
Wallnut Blocks zombies.
Lychee Bomb Explodes and damages zombies in a 9x9 square.
Zombie Appearance Description
Regular Just a regular zombie.
Buckethead A tankier zombie! Loses its bucket helmet at 80% health and its head at 20%.
Newspaper Speeds up after its newspaper is destroyed.
